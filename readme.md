# 简易以太坊实验 (Simple Ethereum Simulator)

本项目是一个简化版的以太坊区块链系统，旨在帮助学生通过实验代码补充的方式，深入彻底地了解区块链的核心底层架构和运作原理。

## 📁 项目结构 (Project Structure)

核心模块和目录结构如下：

- **`block/`**: 包含区块 (Block)、区块链 (Blockchain) 的定义以及交易打包、验证的上链引擎逻辑。
- **`consensus/`**: 包含共识模块，目前涵盖实验所需的 工作量证明 (PoW) 与 权益证明 (PoS)。
- **`crypt/`**: 仅提供基础密码学组件，包含椭圆曲线加密签名 (ECC) 的生成验证以及默克尔树 (Merkle Tree) 获取。
- **`types/`**: 定义了系统的公共基础数据结构，例如账户 (Account)、世界状态 (State)、交易体 (Transaction) 等。
- **`wallet/`**: 包含客户端级别的轻量级简易 SPV 钱包逻辑集成。
- **`main.go`**: 主程序入口文件，拼装并对外暴露出命令行交互操作功能。
- **`实验文档.md`**: **必读！** 包含了所有实验的详细背景、目标要求与代码填空说明。
- **`test.sh`**: 实验测试脚本，校验实验完成是否正确。

## 🎯 实验主线 (Experiment Workflow)

为运行体验完整的区块链功能，你必须先参考同级目录下的《`实验文档.md`》，按步骤手动补全以下三大实验所缺失的核心代码：

1. **实验一：密码学与架构实现**（对应 `crypt/` 下的椭圆曲线及默克尔树构建核心）
2. **实验二：交易引擎与区块上链**（对应 `block/` 的区块验证与挖矿组装逻辑）
3. **实验三：共识算法模块**（对应 `consensus/` 下的 PoW / PoS 计算填空）

## 🧪 验证与自测 (Testing)

填空过程中，不需要等到完全写完，您可随时运行配套的自动化单元测试入口来验证自身代码的正确性：

```bash
# 赋予脚本执行权限
chmod +x test.sh

# 运行自动化校验脚本
./test.sh lab1
./test.sh lab2
./test.sh lab3
```

## 🚀 最终构建 (Build & Run)

当您**完成全部三个实验，并确保 `test.sh` 脚本的最终校验全部通过后**，便可以独立将其打包并在终端级联运行。

请在项目根目录（含有 `go.mod` 的同一层级）依次执行如下构建指令：

### 1. 刷新及同步依赖包
```bash
# 下载并校验所缺少的外部依赖包（推荐配置国内镜像源如 goproxy.cn 加速）
go mod tidy
```

### 2. 编译为可执行文件
```bash
# 将项目构建为一个二进制程序 (此命令将生成 simple_eth.exe)
go build -o simple_eth.exe main.go
```

### 3. 运行交互体验
```bash
# 启动该程序测试帮助指南或体验您的区块链网络
./simple_eth.exe -help
```
